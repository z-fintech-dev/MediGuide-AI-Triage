# 🩺 MediGuide AI: Professional Triage System

An adaptive medical triage application built with Streamlit, designed to bridge the gap between Western clinical care and Traditional natural remedies.

## 🚀 Key Features
- **Smart Triage:** Real-time emergency detection for high-risk symptoms (Chest pain, self-harm, severe infections).
- **Specialist Switching:** Context-aware search across General Physician, Dermatologist, and Psychiatrist databases.
- **Treatment Preferences:** Toggle between Integrated, Western-only, or Traditional-only remedy displays.
- **Atomic Keyword Engine:** High-sensitivity search logic for flexible user input.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Framework:** Streamlit
- **Deployment:** Streamlit Cloud / GitHub
